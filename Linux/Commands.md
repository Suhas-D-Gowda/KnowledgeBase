# Linux Day 1

## Commands Learned

1. pwd
- Shows the path of the current working directory.

2. ls
- Lists files and folders in the current directory.

3. cd <folder_name>
- Moves into the specified directory.

4. cd ..
- Moves one level up in the directory structure.

5. cd
- Takes you directly to the home directory.

6. mkdir <folder_name>
- Creates a new directory.

7. touch <filename>
- Creates an empty file.

8. rm <filename>
- Permanently deletes the specified file.

9. cp <source> <destination>
- Creates a copy of a file or folder.

Example:
cp main.c backup_main.c


10. mv <source> <destination>
- Moves or renames a file or folder.

Examples:
mv mian.c main.c
mv main.c Day2/
