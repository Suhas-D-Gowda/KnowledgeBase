# Git Basics

## What is Git?

Git is a distributed version control system.

It keeps track of changes made to files over time, allowing you to save versions of your work, collaborate with others, and restore previous versions when needed.

---

## Why use Git?

- Track changes
- Restore previous versions
- Collaborate with others
- Experiment safely using branches
- Keep a history of your project

---

## Repository

A repository (repo) is a folder that Git manages.

It contains your project files and the hidden `.git` folder that stores Git's history.
